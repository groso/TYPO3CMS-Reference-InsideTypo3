Browsing the global configuration
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

The **SYSTEM > Configuration** module lets you browse all
the values in the :code:`$GLOBALS['TYPO3_CONF_VARS']` array.

.. figure:: ../../../Images/ConfigurationModule.png
   :alt: Viewing configuration values in the Configuration module


.. note::

   This module is purely a browser. It does not let you change
   values.

   It also lets you browse a number of other global arrays.
   Just be curious and investigate!
